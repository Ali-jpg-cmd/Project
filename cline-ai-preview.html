How do I implement a dark mode toggle in React?
To implement a dark mode toggle in React, you can use the useState hook to track the current theme state and then apply different CSS classes or styles based on that state.

Here's a simple implementation:

1. Create a state variable:
const [darkMode, setDarkMode] = useState(false);

2. Create a toggle function:
const toggleDarkMode = () => setDarkMode(!darkMode);

3. Apply conditional classes to your app container:
<div className={`app ${darkMode ? 'dark' : 'light'}`}>

4. Create a button for toggling:
<button onClick={toggleDarkMode}>Toggle Theme</button>
